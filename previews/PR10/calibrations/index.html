Calibrations

Here we implement functions for deriving the absolute magnitude of the TRGB from various literature sources.

TRGBDistances.Rizzi2007 — Function
Rizzi2007(color, obssym::Symbol, magsym::Symbol, colorsym::Symbol)

Color-dependent TRGB zeropoint magnitude from Rizzi et al. [6].

Supported filter combinations are:

obssymmagsymcolorsymEquation
:JC:I:VI-4.05 + 0.217 * (color - 1.60)
:ACS:F814W:F606W-4.06 + 0.200 * (color - 1.23)
:ACS:F814W:F555W-4.06 + 0.150 * (color - 1.74)
:WFPC2:F814W:F555W-4.01 + 0.180 * (color - 1.58)
:WFPC2:F814W:F606W-4.01 + 0.150 * (color - 1.12)

Returns

  • Float64: The TRGB zeropoint magnitude in the requested band.

Examples

julia> TRGBDistances.Rizzi2007(1.5, :ACS, :F814W, :F606W) ≈ -4.006
true
source
TRGBDistances.Fusco2012 — Function
Fusco2012(color)

Color-dependent TRGB zeropoint magnitude from Fusco et al. [8], $M_{F814W} = -3.63 + 0.08 * (F475W - F814W)^2 - 0.4 * (F475W - F814W)$.

julia> TRGBDistances.Fusco2012(1.5) ≈ -4.05
true
source

:WFC3:F814W:F606W-4.024 + 0.150*(color - 1.1)^2 - 0.050*(color - 1.1)
:WFC3:F814W:F555W-4.019 + 0.113*(color - 1.6)^2 - 0.048*(color - 1.6)
:WFPC2:F814W:F555W-4.007 + 0.123*(color - 1.5)^2 - 0.051*(color - 1.5)
:WFPC2:F814W:F606W-3.984 + 0.179*(color - 1.0)^2 - 0.034*(color - 1.0)

TRGBDistances.Freedman2020 — Function
Freedman2020(color::Union{Real, Nothing}=nothing,
             magsym::Symbol=:I)

TRGB zeropoint calibration from Freedman et al. [11].

Supported parameter combinations are:

magsymAssumed color conventionEquation
:Inone-4.047
:F814Wnone-4.054
:JJ - K-5.14 - 0.85 * (color - 1)
:HJ - K-5.95 - 1.62 * (color - 1)
:KJ - K-6.14 - 1.85 * (color - 1)
:VV - I-2.25 + (color - 1.8)

Arguments

  • color: Optional color term used for color-dependent calibrations. For :J, :H, and :K, the assumed color is J - K. For :V, the assumed color is V - I.
  • magsym: Magnitude/filter band in which the TRGB zeropoint is returned.

Returns

  • Float64: The TRGB zeropoint magnitude in the requested band.

Examples

julia> TRGBDistances.Freedman2020(nothing, :I) ≈ -4.047
true

julia> TRGBDistances.Freedman2020(1.2, :J) ≈ -5.31
true
source

TRGBDistances.Cerny2021 — Function
Cerny2021(color::Union{Real, Nothing}=nothing,
           magsym::Symbol=:I,
           colorsym::Union{Symbol, Nothing}=nothing)

TRGB zeropoint calibration from Cerny et al. [13].

Supported parameter combinations are:

magsymcolorsymEquation
:Inothing-4.056
:F814Wnothing-4.056
:Vnothing-2.26
:Jnothing-5.16
:Hnothing-6.03
:Knothing-6.16
:I:VI-4.06
:F814W:VI-4.06
:V:VI-2.26 + (color - 1.8)
:J:JK-5.16 - 0.85 * (color - 1)
:H:JK-6.03 - 1.72 * (color - 1)
:K:JK-6.16 - 1.85 * (color - 1)

Arguments

  • color: Optional color term used for color-dependent calibrations.
  • magsym: Magnitude/filter band in which the TRGB zeropoint is returned.
  • colorsym: Symbol describing the color combination associated with color.

Returns

  • Float64: The TRGB zeropoint magnitude in the requested band.

Examples

julia> TRGBDistances.Cerny2021() ≈ -4.056
true

julia> TRGBDistances.Cerny2021(1.2, :J, :JK) ≈ -5.33
true
source
TRGBDistances.Hoyt2018 — Function
Hoyt2018(color::Real, magsym::Symbol=:K, colorsym::Symbol=:JK)

TRGB zeropoint from Hoyt et al. [14]. Supported parameter combinations are:

magsymcolorsymAssumed colorEquation
:J:JKJ - K-5.14 - 0.85 * (color - 1)
:H:JKJ - K-5.94 - 1.62 * (color - 1)
:K:JKJ - K-6.14 - 1.85 * (color - 1)
:J:JHJ - H-5.13 - 1.11 * (color - 0.8)
:H:JHJ - H-5.93 - 2.11 * (color - 0.8)
:K:JHJ - H-6.13 - 2.41 * (color - 0.8)

Arguments

  • color: Color term used in the calibration relation. The interpretation depends on colorsym.
  • magsym: Magnitude/filter band in which the TRGB zeropoint is returned.
  • colorsym: Symbol specifying the color convention associated with color.

Returns

  • Float64: The TRGB zeropoint magnitude in the requested band.
julia> TRGBDistances.Hoyt2018(1.5, :K) ≈ -6.14 - 1.85 * (1.5 - 1.0)
true
